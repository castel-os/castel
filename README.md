# Castel

## What is Castel?

Castel is a general-purpose Operating System, built around the idea of be a new community for Free Software, whether you are experienced or not. We don't want to be a system that does everything; we want to do one thing, do it right, and in the best way possible.

## What to expect?

I am not the best programmer and I don't know everything, but I want to do my best and learn to make Castel a reality. We want to build our own tools and solutions to bring the best to our users. Don't expect a Linux-style, Windows-style, or Unix-style or other style system; we are trying to create our own style with the tools we have. The project will start immature, but I hope it will get better over time.
